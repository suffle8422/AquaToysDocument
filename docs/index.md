# AquaToys
このページはAquaToysのサポートページです。

## AquaToysとは
AquaToysはスイマーのとしての活動をより便利にする、さまざまなツールを提供するiOS向けアプリです。
以下を含む様々な機能を提供しています。
- 試合結果管理機能
- 試合結果画像出力機能
- トレーニングメニュー管理機能
- トレーニングメニュー画像出力機能
